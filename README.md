# Trueke - Web App

## Descripción
Proyecto de aplicación web responsiva (PWA en el futuro) para la plataforma Trueke. Desarrollada y mantenida con React y Vite.

## Requisitos Previos
- Node.js (v18 o superior recomendado)
- NPM o Yarn

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/rosieschome/trueke.git
   ```
2. Instala las dependencias (una vez inicializado el proyecto):
   ```bash
   npm install
   ```

## Desarrollo Local

Para arrancar el entorno de desarrollo:
```bash
npm run dev
```

## Compilación para Producción

Para generar una build optimizada para producción:
```bash
npm run build
```
La carpeta resultante será `dist/`.
